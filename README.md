
### Recursively_list_files_s3   

Install the latest Boto 3 release via pip:

$ pip install boto3


Commandline Args:
   file_type: file type eg- .py or .tar.gz

## Usage 

$ python3 list_files_s3.py .tar.gz

- update <Update access keys> and  <update secret keys>
